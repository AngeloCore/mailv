# mailv - Email Validator

A simple module to validate an e-mail address

## Installation

npm:

```sh
npm install mailv

```

yarn:

```sh
yarn add mailv
```

## Usage

#### JavaScript

```js
const mailv = require("mailv");

mailv("test@email.com"); // true
```

#### TypeScript

```ts
import mailv from "mailv";

mailv("test@email.com"); // true
```
